# 🌱 No Waste Tracker

Уеб приложение за намаляване на хранителните отпадъци, създадено за Hack TUES 12 - "Code to Care".

## 📋 Описание на проекта

No Waste Tracker помага на хората да следят храната, която имат у дома, да виждат кои продукти изтичат скоро и да получават идеи как да ги използват навреме, вместо да ги изхвърлят.

### 🎯 Цели на проекта

- **Намаляване на хранителните отпадъци** - чрез проследяване на сроковете на годност
- **Спестяване на пари** - чрез оптимизиране на потреблението
- **Екологично въздействие** - чрез намаляване на въглеродния отпечатък
- **Образоване** - чрез съвети и рецепти за устойчиво потребление

## 🚀 Основни функционалности

### 📦 Управление на продукти
- Добавяне на продукти с име, категория, количество и срок на годност
- Автоматично определяне на статус (безопасно, изтича скоро, изтекъл)
- Маркиране на продукти като използвани или изхвърлени

### ⏰ Напомняния и предупреждения
- Секция "Използвай днес" за спешни продукти
- Цветово кодиране по статус
- Автоматично обновяване на статусите

### 🍳 Интелигентни рецепти
- Предложения за рецепти базирани на налични продукти
- Процент на съвпадение с наличните съставки
- Филтриране по трудност

### 📊 Статистики и Eco Score
- Детайлна статистика за използвани и изхвърлени продукти
- Eco Score система за измерване на екологичното въздействие
- Графики и визуализации с Chart.js

### 🎨 Потребителски интерфейс
- Модерен, тъмен дизайн със зелени акценти
- Responsive дизайн за мобилни устройства
- Плавни анимации и интерактивни елементи

## 🛠️ Технологии

### Frontend
- **HTML5** - семантична структура
- **CSS3** - модерен дизайн с CSS Grid и Flexbox
- **Vanilla JavaScript** - без зависимости от фреймуърци

### Backend & Database
- **Firebase Firestore** - NoSQL база данни в реално време
- **Firebase Hosting** - бърз и надежден хостинг
- **Firebase Authentication** - (подготовка за бъдеща версия)

### Допълнителни библиотеки
- **Chart.js** - за графики и визуализации
- **Firebase SDK** - за комуникация с базата данни

## 🚀 Бърз старт

### 1. Настройка на Firebase

Следвайте упътването в `FIREBASE_SETUP.md` за да:
- Създадете Firebase проект
- Активирате Firestore Database
- Копирате конфигурацията

### 2. Локално тестване

1. Клонирайте проекта
2. Отворете `index.html` в браузър
3. Следвайте инструкциите за настройка на Firebase

### 3. Deploy

```bash
# Инсталирайте Firebase CLI
npm install -g firebase-tools

# Логнете се
firebase login

# Инициализирайте хостинг
firebase init hosting

# Deploy
firebase deploy
```

## 🌍 Социално въздействие

### Проблемът
- 1.3 милиарда тона храна се губят годишно
- 1/3 от цялата храна се изхвърля
- 8% от глобалните емисии са от хранителни отпадъци

### Решението
- Проследяване на сроковете на годност
- Интелигентни напомняния
- Образоване чрез рецепти и съвети
- Визуализация на въздействието
