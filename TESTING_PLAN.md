# 🧪 План за тестване на No Waste Tracker

## 📋 Общ преглед

Този план за тестване покрива всички функционалности на No Waste Tracker преди демонстрация пред журито на Hack TUES 12.

---

## 🔧 **Подготовка за тестване**

### **1. Настройка на средата**
- [ ] Следвайте `FIREBASE_SETUP.md` стъпка по стъпка
- [ ] Уверете се, че всички Firebase конфигурации са правилни
- [ ] Проверете, че Firestore е в "Test mode"
- [ ] Отворете `index.html` в Chrome/Firefox браузър
- [ ] Отворете Developer Console (F12) за гледане на грешки

### **2. Базови проверки**
- [ ] Проверете за JavaScript грешки в конзолата
- [ ] Уверете се, че всички CSS файлове се зареждат
- [ ] Проверете, че Firebase SDK се зарежда правилно
- [ ] Тествайте навигацията между страниците

---

## 📱 **Тестове на Landing Page (`index.html`)**

### **Визуални тестове**
- [ ] Проверете, че всички секции се зареждат правилно
- [ ] Тествайте responsive дизайн на различни размери на екрана
- [ ] Проверете анимациите при hover на feature cards
- [ ] Уверете се, че цветовата схема е консистентна

### **Функционални тестове**
- [ ] Тествайте smooth scrolling към "features" секция
- [ ] Кликнете на "🚀 Започни сега" - трябва да отиде към `dashboard.html`
- [ ] Кликнете на "📖 Научи повече" - трябва да скролира надолу
- [ ] Кликнете на "🌱 Започнете днес" - трябва да отиде към `dashboard.html`

### **Тестове на мобилна версия**
- [ ] Намалете прозореца до мобилни размери (<768px)
- [ ] Проверете, че бутоните се подреждат вертикално
- [ ] Тествайте, че текстът е четлив на малък екран
- [ ] Проверете, че всички елементи са видими без хоризонтален скрол

## 📊 **Тестове на Dashboard (`dashboard.html`)**

### **Зареждане и базови функционалности**
- [ ] Отворете страницата и проверете, че се зарежда без грешки
- [ ] Проверете, че summary cards се показват с нули в началото
- [ ] Тествайте, че "Use Today" секция се показва със съобщение за празно състояние
- [ ] Проверете, че филтър бутоните работят

### **Тестове с данни**
- [ ] Отидете на `add-product.html` и добавете тестови продукти
- [ ] Върнете се към dashboard и проверете, че продуктите се появяват
- [ ] Тествайте филтрирането по статус (all, safe, use-soon, expired)
- [ ] Проверете, че броячите в summary cards се обновяват правилно

### **Тестове на продукт карти**
- [ ] Кликнете на "✅ Използван" - проверете, че продуктът се маркира
- [ ] Кликнете на "❌ Изхвърлен" - проверете, че продуктът се маркира
- [ ] Проверете, че Eco Score се обновява след маркиране
- [ ] Тествайте, че статистиките се обновяват автоматично

### **Тестове на статусове**
- [ ] Добавете продукт със срок днес - трябва да се покаже като "use-soon"
- [ ] Добавете продукт със срок вчера - трябва да се покаже като "expired"
- [ ] Добавете продукт със срок след 5 дни - трябва да се покаже като "safe"
- [ ] Проверете, че цветът на статусите е правилен (зелен/оранжев/червен)

## 📈 **Тестове на Статистики (`stats.html`)**

### **Съвети за подобрение**

   - [ ] Тествайте филтрирането
   - [ ] Проверете "Use Today" секцията

4. **Маркирайте продукти**

   - [ ] Отидете на `stats.html`
   - [ ] Проверете Eco Score и графики

6. **Тествайте рецепти**
   - [ ] Отидете на `recipes.html`
   - [ ] Намерете подходяща рецепта
   - [ ] "Сгответе" рецептата и проверете резултата
